A simple flash card utility
---------------------------

Only arg is filename

File structure:
---------------

>n (num of cards)

>string 1a (front of card)

>string 1b (back of card)

>string 2a

>string 2b

>...

>string na

>string nb

Example:
--------

>3

>Capital of Hungary

>Budapest

>Number of letters in alphabet

>Around 12

>Letters in the word "potato"

>six
